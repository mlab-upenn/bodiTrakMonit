Detailed Description

These functions are used to communicate with the T7 sensor array.

Function Documentation

FSAT7 fsat7Create ( FSAT7ID  t7id,
int  loadCal 
)

Creates an FSAT7 object and optionally downloads it's calibration data. Call fsat7Destroy for each successful call to this function.

Parameters
t7idan FSAT7ID that identifies which T7 to connect to; set to zero if any T7 will do.
loadCalif non-zero the T7 calibration data is downloaded from the mat and all fsat7Scan values will be calibrated; otherwise fsat7Scan values will be conductance readings.
Returns
a reference to the FSAT7 object; to be used with all other fsat7 functions.

FSAT7ID* fsat7CreateList ( )

Creates a list of T7 sensor arrays connected to the computer. Use fsat7Create to open a connection to one of the items in the list. The list is terminated with a null FSAT7ID. Call fsat7DestroyList for each successful call to this function.

a pointer to an array of FSAPORTIDs.

int fsat7Scan ( FSAT7  t7,
float *  values,
int *  flags 
)

Scans the sensor array from left to right, top to bottom and returns the scanned values.

If the calibration has been loaded then the returned values are calibrated readings; otherwise the values are conductances (measured in microSiemens).

Parameters
t7the T7 object.
valuesa pointer to a buffer that will hold the scanned values.
flagsa pointer to a buffer that will hold the flags for each scanned value.
Returns
the number of scanned sensors.
